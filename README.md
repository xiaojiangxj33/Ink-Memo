# Ink Notes - 墨水屏笔记应用

## 项目概述

Ink Notes 是一个功能完整的墨水屏笔记应用，整合了蓝牙连接、设备控制和蓝牙传图功能，支持多种文字处理和绘图功能。

## 已完成功能模块

### 蓝牙连接模块
- ✅ 蓝牙设备扫描和连接
- ✅ 蓝牙连接状态管理
- ✅ 连接断开和重连处理

### 设备控制模块
- ✅ 设备初始化
- ✅ 屏幕清除功能
- ✅ 时间同步
- ✅ 设备模式设置（日历模式、时钟模式）

### 蓝牙传图模块
- ✅ 图片上传和预览
- ✅ 多种图像抖动算法（Floyd-Steinberg、Atkinson、Bayer、Stucki、Jarvis-Judice-Ninke）
- ✅ 图片数据转换
- ✅ 图片数据发送

### 画布绘图模块
- ✅ 文字添加和编辑
- ✅ 画布缩放和移动

### 文字处理功能
- ✅ 文字大小独立控制
- ✅ 文字颜色修改
- ✅ 文字自动换行
- ✅ 文字拖动和删除

### UI优化
- ✅ 赛博朋克风格一致性
- ✅ 蓝牙控制面板滑动功能
- ✅ 字体样式统一
- ✅ 按钮样式统一

## 本地运行

### 前置条件
- Node.js

### 运行步骤
1. 安装依赖：
   ```bash
   npm install
   ```
2. 运行应用：
   ```bash
   npm run dev
   ```

## 构建项目

```bash
npm run build
```

## 文件结构

```
ink-notes/
├── components/          # React组件
├── bluetooth/          # 蓝牙功能相关代码
│   ├── bluetooth.ts    # 蓝牙连接和通信逻辑
│   ├── canvas.ts       # 画布处理和绘图逻辑
│   ├── dithering.ts    # 图像抖动处理
│   └── types.ts        # 蓝牙相关类型定义
├── index.tsx           # 应用主入口（整合蓝牙功能）
├── index.css           # 应用样式（整合蓝牙样式）
└── ...
```
